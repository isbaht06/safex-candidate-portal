# SafeX Intern Candidate Management Portal

A candidate directory feature built for SafeX Solution internal HR tooling, showcasing search, status filtering and a table based candidate view styled to match the SafeX brand.

## Tech Stack

- Frontend: React + Vite
- Backend: .NET Web API (in progress)
- Version Control: GitHub

## Features

- Search candidates by name or role
- Filter by application status (dropdown + quick-filter tabs)
- Paginated candidate table with avatar initials, status badges and contact info
- Row-level actions: view profile, email candidate, view resume (placeholders pending backend)
